# Light and Textures
Autor: Yone Hernández León

### Descripción de la aplicación
La aplicación consiste en una recreación de un pequeño set de muebles y figuras, las cuales serán iluminadas con una luz para ver como sus  brillos y colores cambian.

### Caracteristicas de la aplicación
La aplicación otorga la capacidad de:
- Rotar el set de figuras a izquierda y derecha
- Iluminar el set de figuras
- Cambiar las tres tonalidades (RGB) de la luz que flashea al set de figuras
- Resetear el set de figuras a su estado inicial

### Decisiones adoptadas para la solución propuesta
Al ser la aplicación un proyecto más bien sencillo, no se hizo uso de la programación con clases y objetos. Primero se programó el diseño del set de figuras a iluminar. Debido a que las figuras no fueron creación propia, en esta parte del desarrollo surgieron grandes problemas que se lograron subsanar correctamente. Luego se añadió la rotación correcta al set y se le añadio la luz ambiente al mismo. Continuó el desarrollo con la programación de la luz que se lanza desde el frente al set, así como la modificación de las tonalidades de esta luz. Finalmente, se modificaron algunos atributos a las figuras del set (como brillo y ambiente) para diferenciarlas un poco entre sí, así como se creó el menú y se corrigieron errores finales
